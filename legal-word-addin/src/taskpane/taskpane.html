Anonymize

Redact replaces each hit with [REDACTED]. Fake uses random fictitious names, companies, and addresses. For people, the same first or last token keeps one fake (e.g. “Jane Doe”, “Doe”, “Ms. Doe” line up). Skips times, prices, and routine contract dates.

Translate

Selection translates only what is highlighted (same BIU / tracked rules when Word exposes paragraph markup). Document uses the same range + bookmark + single insertOoxml approach as selection, but splits the body into chunks (several paragraphs per chunk) and applies chunks last→first so indices stay stable. API calls stay within about 2.5k input tokens each. Progress fills the translate button. Other character formatting is not preserved.

Just describe what you need — the AI sees both your selection and the full document, and will figure out the scope and action. Changes are applied as tracked revisions.
